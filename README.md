## Installation

### CocoaPods

PrettyWaterfallCollectionViewLayout is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'PrettyWaterfallCollectionViewLayout'
```

## Requirements

- **iOS 10.0** or higher
- **Xcode 9.0** or higher
